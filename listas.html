Lenguajes para el desarrollo web

HTML
HyperText Markup Lenguage: Es un lenguaje de marcado que se usa para estructurar datos en una página web
CSS
Cascade Style Sheets: Es un lenguaje de diseño que se utiliza para dar estilos al HTML
JavaScript
Es un lenguaje de programacion para dar interactividad a un sitio web
